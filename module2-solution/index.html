Web Designing

HTML

HTML stands for Hyper Text Markup Language.HTML is the standard markup language for creating Web pages.HTML describes the structure of Web pages.

CSS

CSS stands for Cascading Style Sheets.CSS describes how Html elements are to be diplayed on screen.External stylesheets are stored in CSS files.

JavaScript

JavaScript is the Programming Language for the Web.JavaScript can change both HTML and CSS.JavaScript can calculate,manipulate and validate data.
